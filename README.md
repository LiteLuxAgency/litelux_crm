# LiteLux CRM Mini App

Быстрый MVP CRM под Telegram Mini App:

- главная страница со списком клиентов;
- кнопка `+` справа снизу для добавления;
- шестеренка сверху справа для настроек;
- поля: имя, телефон, комиссия, дубль, эксклюзив, ссылка, только клиенты, перезвонить, нет ответа;
- уведомления в Telegram по точной дате и времени;
- повторное напоминание через 24 часа при галочке `Нет ответа`.

## Стек

- `apps/web`: React + Vite + TypeScript
- `apps/server`: Express + TypeScript + Telegraf + Supabase
- [`supabase/schema.sql`](/Users/jgb/LiteLux/БД/supabase/schema.sql): SQL для Supabase

## Запуск

1. Создайте проект в Supabase.
2. Выполните SQL из файла [`supabase/schema.sql`](/Users/jgb/LiteLux/БД/supabase/schema.sql).
3. Создайте `.env` на основе [`.env.example`](/Users/jgb/LiteLux/БД/.env.example).
4. Установите зависимости:

```bash
npm install
```

5. Запустите проект:

```bash
npm run dev
```

Фронтенд поднимется на `http://localhost:5173`, сервер на `http://localhost:3000`.

## Настройка Telegram-бота

1. Создайте бота через `@BotFather`.
2. Откройте Mini App, зайдите в настройки.
3. Вставьте токен бота, включите бота и сохраните настройки.
4. После сохранения откройте ссылку на бота из настроек и нажмите `Start`.
5. Бот сохранит текущий чат, и туда пойдут уведомления о перезвоне.

## Как работают напоминания

- `Перезвонить`: когда наступает дата и время, бот отправляет уведомление.
- `Нет ответа`: сервер ставит отдельное напоминание через 24 часа.
- Проверка напоминаний работает внутри сервера по интервалу `REMINDER_CHECK_INTERVAL_MS`.
